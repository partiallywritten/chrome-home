## Chrome Home
New tab extension for chrome that I use.

### Install
- Download the [source code](https://github.com/partiallywritten/chrome-home/archive/refs/heads/main.zip) and unzip it.
- Head to `chrome://extensions` and enable developer mode.
- Click load unpacked and select the unzipped folder.

### Warning
This entire extension has been fully vibe-coded using [copilot](https://github.com/apps/copilot-swe-agent).
